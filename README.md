## 📘 Course Overview

รายวิชา **Computer Programming for Computer Engineer (ENGCE117)**  
มุ่งเน้นการพัฒนาทักษะการเขียนโปรแกรมเชิงลึกในระดับวิศวกรรมคอมพิวเตอร์  
ครอบคลุมทั้งโครงสร้างข้อมูล อัลกอริทึม การจัดการหน่วยความจำ และการออกแบบระบบอย่างเป็นระบบ

---

## 🧠 Topics Covered

### 🔹 Memory & Pointer
- Pointer ขั้นสูง
- Pointer-to-pointer
- Dynamic Memory Allocation (malloc / free)
- Memory Safety

### 🔹 Data Structure
- Array
- Structure
- Singly Linked List
- Doubly Linked List
- Node Manipulation

### 🔹 File Handling
- File Input / Output
- ASCII File Processing
- Data Persistence

### 🔹 Recursion
- ฟังก์ชันเรียกตัวเอง
- Stack Behavior
- Divide-and-Conquer Concept

### 🔹 Object-Oriented Programming
- Encapsulation
- Inheritance
- Polymorphism
- Abstract Class

### 🔹 Algorithms
- Divide-and-Conquer
- Binary Search O(log n)
- Merge Sort O(n log n)
- Greedy Algorithm
- Dynamic Programming
- Backtracking

### 🔹 Numerical Method
- Bisection Method

### 🔹 Software Engineering Practice
- Code Structuring
- Clean Code Principle
- Software Testing
- Git Version Control

## 🎯 Learning Outcome

หลังจบรายวิชา นักศึกษาสามารถ:

- ออกแบบโครงสร้างข้อมูลได้อย่างถูกต้อง
- วิเคราะห์ความซับซ้อนของอัลกอริทึม
- จัดการหน่วยความจำได้อย่างปลอดภัย
- เขียนโปรแกรมเชิงวัตถุได้อย่างมีมาตรฐาน
- พัฒนาโค้ดที่อ่านง่ายและบำรุงรักษาได้
