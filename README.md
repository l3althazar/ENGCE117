## 📘 คำอธิบายรายวิชา | Course Description

รายวิชา **Computer Programming for Computer Engineer (ENGCE117)**  
ศึกษาและฝึกปฏิบัติหลักการเขียนโปรแกรมเชิงลึก ครอบคลุมการจัดการหน่วยความจำ โครงสร้างข้อมูล อัลกอริทึม และการออกแบบโปรแกรมเชิงวัตถุ

This course focuses on advanced programming concepts for computer engineers, including memory management, data structures, algorithms, and object-oriented design principles.

---

# 📚 เนื้อหาที่ศึกษา | Topics Covered

---

## 🔹 1. ฟังก์ชันและการส่งค่าผ่านอาเรย์  
**Functions & Array Parameter Passing**

- Pass by value  
- Pass by reference  
- Dynamic array usage  

---

## 🔹 2. ฟังก์ชันกับโครงสร้างข้อมูล (Struct)  
**Functions with Structures**

- Structure declaration  
- Passing struct to functions  
- Memory layout understanding  

---

## 🔹 3. การเปลี่ยนตำแหน่งการชี้ของพอยน์เตอร์  
**Pointer Manipulation**

- Pointer arithmetic  
- Pointer-to-pointer  
- Address control  

---

## 🔹 4. การจัดการลิงก์ลิสต์แบบทางเดียว  
**Singly Linked List**

- Node creation  
- Insert / Delete  
- Traversal  
- Dynamic memory allocation  

---

## 🔹 5. การจัดการลิงก์ลิสต์แบบสองทาง  
**Doubly Linked List**

- Bidirectional traversal  
- Node linking logic  
- Memory safety  

---

## 🔹 6. การเขียนโปรแกรมเชิงวัตถุเบื้องต้น  
**Introduction to Object-Oriented Programming**

- Class & Object  
- Encapsulation  
- Constructor / Destructor  

---

## 🔹 7. การจัดการไฟล์ข้อมูล  
**File Handling**

- File input / output  
- ASCII file processing  
- Data persistence  

---

## 🔹 8. ฟังก์ชันเรียกตัวเอง  
**Recursion**

- Recursive logic  
- Stack behavior  
- Base case & recursive case  

---

## 🔹 9. เทคนิคอัลกอริทึมแบบ Divide-and-Conquer  
**Divide and Conquer Algorithm**

- Binary Search O(log n)  
- Merge Sort O(n log n)  

---

## 🔹 10. เทคนิคอัลกอริทึมแบบ Greedy  
**Greedy Algorithm**

- Knapsack Problem  
- Local optimal strategy  

---

## 🔹 11. เทคนิคอัลกอริทึมแบบ Dynamic Programming  
**Dynamic Programming**

- Optimal substructure  
- Overlapping subproblems  
- Knapsack optimization  

---

## 🔹 12. เทคนิคอัลกอริทึมแบบ Backtracking  
**Backtracking**

- State space tree  
- Constraint solving  

---

## 🔹 13. วิธีการเชิงตัวเลข (Bisection Method)  
**Numerical Methods**

- Root finding  
- Iterative approximation  

---

# 🛠 ทักษะที่พัฒนา | Skills Developed

- Algorithm Analysis  
- Clean & Modular Code Design  
- Dynamic Memory Control  
- Debugging & Logical Thinking  
- Version Control with Git  
- Engineering Problem Solving  

---

# 🎯 ผลลัพธ์การเรียนรู้ | Learning Outcome

หลังจบรายวิชา สามารถ:

- ออกแบบโครงสร้างข้อมูลได้อย่างมีประสิทธิภาพ  
- วิเคราะห์ความซับซ้อนของอัลกอริทึม  
- จัดการหน่วยความจำได้อย่างปลอดภัย  
- เขียนโปรแกรมเชิงวัตถุได้อย่างเป็นระบบ  

Students are able to design efficient data structures, analyze algorithm complexity, manage memory safely, and implement structured object-oriented programs.
